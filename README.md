# Team Task Manager

This state-of-the-art Team Task Manager is a comprehensive, full-stack workflow orchestration platform engineered with Next.js and Supabase that empowers organizations to manage complex projects through a robust role-based access control (RBAC) system, featuring a modern, minimalist light-themed interface enhanced with Lucide-react iconography and a responsive dashboard layout. The application enables administrators to maintain absolute oversight via the project management suite at `/api/projects` (GET/POST) and `/api/projects/[id]` (PATCH/DELETE) while orchestrating task lifecycles through the granular task engine at `/api/tasks` (GET/POST) and `/api/tasks/[id]` (PATCH/DELETE), which uniquely supports multi-user assignments via a sophisticated many-to-many junction architecture that restricts task visibility and status-toggle permissions exclusively to assigned members and administrators to ensure data privacy and operational focus. Users navigate through a seamless front-end experience starting from the landing page (`/`), proceeding through high-security authentication gates at the login (`/login`) and signup (`/signup`) routes—powered by JWT-verified backend endpoints at `/api/auth/login` and `/api/auth/signup`—ultimately arriving at the centralized dashboard (`/dashboard`), where real-time statistics summarize workspace health, a dynamic sidebar facilitates project-filtered navigation, and an intuitive chip-based multi-selection UI allows for precise task delegation to team members fetched via the `/api/users` directory.
